te_algorithms documentation
===========================

`te_algorithms` is a Python package containing the code for the algorithms used
by the Trends.Earth QGIS Plugin and related tools. It contains code for
calculating land degradation status and trends (in support of SDG Indicator
15.3.1), as well as statistics related to drought (following the
recommendations for assessing UNCCD Strategic Objective 3)
